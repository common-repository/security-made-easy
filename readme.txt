== Description ==

A set and forget solution for WordPress security.

**Features**

* Add cross-site scripting (XSS) protection
* Add a powerful, yet lightweight and hands-free firewall
* Disable directory indexing
* Disable theme/plugin file editing

Ask for help: [help@veryeasy.io](mailto:help@veryeasy.io)

== Installation ==

**Automatic**

* From your WordPress Admin, navigate to: **Plugins** > **Add New**
* Search for: **Security Made Easy**
* Install it
* Activate it

**Manual**

* Download
* Unzip
* Upload to /plugins/ folder
* Activate
